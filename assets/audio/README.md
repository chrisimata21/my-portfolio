## Instructions:
Add more audio files here to expand your music collection.

### To Add More Tracks:
1. Place your audio files in this folder
2. Update the `tracks` array in `script.js` with the new filename and display name

### Recommended Audio Types:
- **Hip-hop/Rap** - Like your current MOJO JOJO track
- **Lo-fi beats** - Relaxing hip-hop instrumentals  
- **Focus music** - Concentration-enhancing tracks
- **Instrumental** - Any genre without vocals for background listening

### File Requirements:
- **Format**: MP3, WAV, or OGG
- **Duration**: 2-10 minutes (they will loop automatically)
- **Volume**: Moderate levels (the player has volume control)
- **Quality**: 128-320 kbps for good quality without large file sizes

### Note:
If audio files fail to load, the music player will fall back to a simple tone generator for demonstration purposes.
